# self

本人の思考パターン・学習スタイル・自己理解に関する知見。

* [「やりたいことがやれない」の構造 — バースト型実行の再起動設計](/self/burst-execution-design.md) - やりたいことが消化できない主因は意志力でなく認知の方向非対称と機構の欠如。streak は貯金型意志力で全損する。連続稼働でなく MTTR ゼロ（ステートレス・強制点・最小動作・常設ランプ）の再起動設計へ。
* [思考の癖の研究 — knowledge/ と自作ハーネスから抽出した認知プロファイル](/self/cognitive-profile.md) - 行動ログ分析とインタビュー3ラウンドによる認知プロファイル。体系化はAI以後の外骨格、自然体は直感・感情・身体系。ハーネス哲学は自己認識の投影、という中心発見。
* [「言語化が下手」の分解と訓練 — 述語は書くものではなく採るもの](/self/verbalization-training.md) - 言語化を4種（命名・説明・手順化・述語化）に分解し、実例聞き取りで「AIへの成功条件の述語化」に確定。白紙生成をやめ、判定筋だけで述語を得る3採取源（AI候補→採点・反例の裏返し・差し戻しからの採掘）と P/J/W 仕分けによる「リストが書けない」の機序解明、最小動作「証拠物1つ指定」。
* [自分を研究する方法カタログ — 質問以外の7つの道具](/self/self-research-methods.md) - 自己研究の方法を「自己申告・痕跡分析・他者報告・実験・投影法・経験サンプリング・測定ゲーム」の7系統に整理。ジョハリの窓で届く領域が違う、が選択の指針。
