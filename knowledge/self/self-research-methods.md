# 問い

「私を研究する方法って質問以外にないか」。[認知プロファイル研究](/self/cognitive-profile.md)の続き。

# 前提: 方法ごとに届く領域が違う（ジョハリの窓）

- 質問（自己申告）が届くのは「自分は知っているが隠れている」領域のみ。
  自己申告には社会的望ましさバイアス・内観の限界（自分の判断過程は自分に見えない）がある。
- **盲点領域**（他人は知るが自分は知らない）→ 他者報告・痕跡分析
- **未知領域**（誰も知らない）→ 実験・投影法・測定ゲーム
- 複数の方法で同じ結論が出たときだけ信じる（триangulation・三角測量）。

# 7つの方法

1. **痕跡分析（デジタル考古学）** — 残そうと思っていないデータほど正直。
   ブラウザ履歴・再生履歴・購入履歴・カメラロール・チャット送信時刻の分布・git log の時間帯。
   「持ち物とブックマークは化石化した意思決定」。※真面目な面に偏る弱点は選ぶデータ源で補正。
2. **他者報告（360度）** — 盲点領域の唯一の入口。家族・友人・同僚に同じ3問を聞く
   （例:「私が自覚してなさそうな癖は？」「私に頼みたくないことは？」「私といて疲れる瞬間は？」）。
3. **実験（予測→観測）** — 研究者（AI）が事前に行動を予言して封をし、実際の選択と照合する。
   予言の的中率がプロファイルの精度そのもの。反証可能性が最も高い方法。
4. **投影法** — 曖昧な刺激（未完の物語・多義的な状況・写真）に解釈を出させる。
   曖昧さを埋めるものが本人の priors。TAT・文章完成法の系譜。
5. **経験サンプリング（ESM）** — ランダムな時点で「今何をしていた・気分は」を短く記録。
   記憶による再構成を通さない現在形のデータ。業務構造化で発明した
   [実況メモ](/tech/business-process-structuring-for-llm.md)の自分適用版。cron/hook で機械化可能。
6. **測定ゲーム** — 遊びの形をした心理測定。キャリブレーションクイズ（推定値+自信%→
   自信過剰/過小を数値化）、時間圧下の二択 vs 熟考の二択の乖離（直感と理性のズレの検出）、
   リスク選好ゲーム。
7. **言語分析** — 本人の書いた文章の統計的な癖（ヘッジ表現の頻度・一人称・文長・
   感情語彙）。内容ではなく形式を見るので自己申告バイアスが乗りにくい。

# 運用のコツ

- 1つの発見は1つの方法で確定させない。質問で立てた仮説を痕跡か実験で検証する往復が本体。
- 予測はすべて事前に書面化する（後付け解釈の防止）。外れた予測こそ記録する —
  [失敗観測駆動](/tech/ai-agent-harness-basics.md)の自己研究版。

# Examples

- 実例: 認知プロファイル研究では痕跡分析（knowledge/・失敗カタログ）→インタビューの順で
  三角測量し、痕跡由来の仮説2件がインタビューで反証された。方法を変えると結論が変わる実証。
